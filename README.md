# Welcome to eTutor

## eTutor for University school system

### Clone the project via

```git/bash
    git clone https://github.com/quannh-netalik/eTutor-fpt.git
```

### FE

Installation:

```git/bash
    cd fe
    yarn
```

Then start the server:

```git/bash
    cd be
    yarn run start
```

### BE

Installation:

```git/bash
    cd be
    yarn
```

Then start the server:

```git/bash
    cd be
    yarn run start
```
